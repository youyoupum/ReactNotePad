## 💡 快速答疑

### Q: 路由配置在哪？
**A**: `src/routes/AppRouter.jsx`

### Q: 如何跳转页面？
**A**: 
```jsx
// 方法1：使用 Link
<Link to="/about">关于</Link>

// 方法2：使用 navigate
const navigate = useNavigate();
navigate('/about');
```

### Q: 如何获取 URL 参数？
**A**:
```jsx
// URL: /notes/123
const { id } = useParams();  // id = "123"
```

### Q: 如何保护页面（需要登录）？
**A**: 使用 `ProtectedRoute` 包裹组件
```jsx
<Route path="/profile" element={
  <ProtectedRoute>
    <ProfilePage />
  </ProtectedRoute>
} />
```
